PageSponge: Queue

The queue is a method of sponging URLs, rather than visiting each URL individually. Input one or more URLs below and press the Bulk Add button to begin. It will open a new tab for each URL, wait for the page to finish loading, and save the results. If the tab is closed, errors, or timeouts [after 60 seconds] then it will retry up to 3 times.
